# 🚀 БЫСТРЫЙ СТАРТ — ИИ-Помощник

## Запуск

### Вариант 1: Через .bat файл (Windows)
```
start_ai.bat
```

### Вариант 2: Через Python
```bash
cd ai_assistant
python main_ai.py
```

### Вариант 3: С опциями
```bash
python main_ai.py --simple   # Простой режим без цветов
python main_ai.py --test     # Тестовый запуск
python main_ai.py --demo     # Демонстрация возможностей
python main_ai.py --help     # Справка
```

## 📖 Основные команды

| Команда | Описание |
|---------|----------|
| `/help` | Показать все команды |
| `/good` | **Обучение**: ответ хороший ✅ |
| `/bad` | **Обучение**: ответ плохой ❌ |
| `/remember текст` | Запомнить факт |
| `/search запрос` | Поиск в памяти |
| `/mood настроение` | Сообщить о настроении |
| `/code тип описание` | Генерация кода |
| `/game запрос` | Помощь с играми |
| `/explain термин` | Объяснить концепцию |
| `/debug код` | Найти ошибку |
| `/stats` | Статистика обучения |
| `/exit` | Выйти |

## 💡 Примеры использования

### 1. Помощь с кодом
```
➤ /code функция суммирует два числа
➤ /explain что такое декоратор
➤ /debug [ваш код с ошибкой]
```

### 2. Разработка игр
```
➤ /game идея
➤ /game квест
➤ /game название персонаж
➤ /game шаблон platformer
```

### 3. Эмоциональная поддержка
```
➤ Я сегодня устал
➤ Мне грустно

```

### 4. Обучение
```
➤ [получили ответ]
```

## 💡 Советы

1. **Используйте `/good` и `/bad`** — ИИ учится на вашем фидбеке
2. **Задавайте конкретные вопросы** — чем точнее запрос, тем лучше ответ
3. **Сохраняйте важные факты** через `/remember`
4. **Проверяйте `/stats`** — смотрите прогресс обучения
5. **Экспериментируйте** — ИИ поддерживает много тем!

## ⚠️ Важно

- Все данные хранятся **локально** в `ai_memory.db`
- ИИ работает **без интернета**
- Нет внешних API — полная приватность
- **Обучается только на вашем фидбеке**
