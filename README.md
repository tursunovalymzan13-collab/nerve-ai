# ИИ-Помощник v1.0

Персональный ИИ-помощник без ограничений для обучения, поддержки и помощи в разработке.

## 🚀 Возможности

### 💻 Кодирование
- Генерация функций, классов, шаблонов кода
- Объяснение концепций программирования
- Поиск и исправление ошибок (дебаггинг)
- Примеры кода по различным темам
- Создание шаблонов проектов (Flask, Django, Telegram-боты, Pygame)

### 🎮 Разработка игр
- Генерация идей для игр
- Создание квестов и заданий
- Генерация названий (персонажи, локации, предметы)
- Шаблоны кода для Pygame (платформер, шутер, базовый)
- Рекомендации игровых движков
- Информация о жанрах и механиках

### 💙 Эмоциональная поддержка
- Анализ настроения по тексту
- Поддерживающие ответы в трудные моменты
- Ободрение и мотивация
- Распознавание эмоций (радость, грусть, злость, тревога, усталость)

### 🧠 Обучение на фидбеке
- Оценка ответов командами `/good` и `/bad`
- Запоминание предпочтений пользователя
- Поиск в истории диалогов
- Адаптация стиля ответов
- Сохранение знаний и фактов

## 📦 Установка

### 1. Установите зависимости

```bash
pip install colorama
```

Или все зависимости:

```bash
pip install -r requirements.txt
```

### 2. Запуск

```bash
# Основной режим (с цветами)
python ai_assistant/main_ai.py

# Простой режим (без цветов)
python ai_assistant/main_ai.py --simple

# Тестовый запуск
python ai_assistant/main_ai.py --test

# Демонстрация возможностей
python ai_assistant/main_ai.py --demo

# Справка
python ai_assistant/main_ai.py --help
```

## 📖 Команды в чате

| Команда | Описание |
|---------|----------|
| `/help` | Показать список команд |
| `/good` | Отметить последний ответ как хороший |
| `/bad` | Отметить последний ответ как плохой |
| `/remember [текст]` | Запомнить информацию |
| `/search [запрос]` | Поиск в памяти |
| `/mood [настроение]` | Сообщить о настроении |
| `/code [тип] [описание]` | Генерация кода |
| `/game [запрос]` | Помощь с разработкой игр |
| `/explain [термин]` | Объяснить концепцию |
| `/debug [код]` | Найти ошибку в коде |
| `/stats` | Показать статистику обучения |
| `/clear` | Очистить экран |
| `/exit` | Выйти из программы |

## 💡 Примеры использования

### Помощь с кодом

```
➤ напиши функцию для сложения чисел
➜ def add_numbers(a, b):
      """Суммирует два числа"""
      result = a + b
      return result
```

### Разработка игр

```
➤ /game идея
➜ 🎮 Идея игры
   Жанр: platformer

   Фишка: гравитация меняется
```

## 🏗️ Архитектура

```
ai_assistant/
├── memory.py      # База данных (SQLite) для памяти
├── learner.py     # Система обучения на фидбеке
├── emotions.py    # Анализ эмоций и эмпатия
├── coder.py       # Помощник по программированию
├── game_dev.py    # Помощник по разработке игр
├── ai_core.py     # Ядро, объединяющее все модули
├── chat_ui.py     # Консольный интерфейс
└── main_ai.py     # Точка входа
```

## 🔧 Модули

### Memory (memory.py)
- Хранение диалогов
- Обратная связь (good/bad)
- Предпочтения пользователя
- База знаний
- Шаблоны кода

### Learner (learner.py)
- Классификация сообщений
- Поиск похожих успешных ответов
- Адаптация стиля
- Статистика обучения

### Emotions (emotions.py)
- Анализ текста на эмоции
- Поддерживающие ответы
- Отслеживание настроения
- Эмодзи для эмоций

### Coder (coder.py)
- Шаблоны кода (Python, JavaScript, SQL, HTML, CSS)
- Паттерны проектирования
- Объяснение концепций
- Дебаггинг
- Оптимизация кода

### GameDev (game_dev.py)
- Информация о жанрах
- Генераторы имён
- Создание квестов
- Баланс игры
- Шаблоны Pygame

## 📝 База данных

ИИ использует SQLite для хранения:

- **conversations** — история диалогов
- **feedback** — оценки ответов (good/bad)
- **preferences** — предпочтения пользователя
- **knowledge** — факты и знания
- **code_patterns** — шаблоны кода

Файл базы: `ai_memory.db` (создаётся автоматически)
